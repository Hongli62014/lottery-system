# 抽奖系统操作说明

## 一、首次使用步骤

### 1. 准备数据
首次进入系统时，默认没有数据展示。您可以选择以下两种方式开始使用：

#### 方式一：使用默认数据
系统提供了默认的示例数据，您可以直接使用这些数据来查看系统的整体展示效果。

#### 方式二：导入自己的数据（推荐）

1. **下载模板**：
   - 进入「配置」页面
   - 点击「人员配置」→「人员名单」
   - 点击「下载模板」按钮，下载Excel数据模板
   - 打开模板，根据示例填写您的人员数据（**请注意：表头不可修改**）

2. **上传数据**：
   - 在同一个页面点击「上传文件」按钮
   - 选择您修改好的Excel表格文件
   - 等待上传完成，系统会自动加载您的人员数据

### 2. 配置奖品

1. 进入「配置」页面 →「奖品配置」
2. 点击「添加奖品」按钮，添加您的奖品信息：
   - **名称**：尽量简短，方便展示
   - **是否全员参加**：
     - 勾选：该奖项从全体人员中抽取（已中奖人员依然可以参与）
     - 不勾选：该奖项从剩余未中奖人员中抽取
   - **获奖人数**：该奖项要抽取的人数
   - **已获奖人数**：系统自动统计，不可编辑
   - **已抽取**：
     - 勾选：该奖项已使用
     - 取消勾选：重置该奖项，但不会重置已获奖人员
   - **图片**：
     - 点击「选择图片」按钮，从图片列表中选择奖品图片
     - 图片列表中的图片可在「配置」→「图片管理」中上传
   - **顺序调整**：点击左侧图标可以调整奖品顺序

3. 完成后，点击「保存」按钮保存奖品配置

## 二、抽奖操作

### 1. 进入抽奖页面
- 点击页面顶部导航栏的「首页」按钮，进入抽奖页面

### 2. 抽奖流程

1. **选择奖项**：在左侧奖项列表中选择要抽取的奖项
2. **开始抽奖**：点击「开始抽奖」按钮，人员名单开始滚动
3. **停止抽奖**：点击「停止抽奖」按钮，系统会随机选中一名中奖人员
4. **查看结果**：
   - 中奖人员会显示在页面中央
   - 中奖信息会自动记录在系统中
   - 已中奖人员会从剩余人员中排除（除非该奖项设置为「全员参加」）

### 3. 临时抽奖
- 在抽奖页面的奖项列表右上角有一个「+」号按钮
- 点击该按钮可以临时增加一个抽奖奖项
- 一次只能增加一项临时抽奖
- 新增成功后，当前奖项会自动设置为该临时奖项
- 抽取成功后，系统会返回正常奖项列表

## 三、高级功能

### 1. 音乐管理

1. 进入「配置」页面 →「音乐管理」
2. **上传音乐**：点击「上传音乐」按钮，上传您喜欢的音乐文件
3. **音乐播放控制**：
   - 鼠标左键点击：播放/暂停
   - 鼠标右键点击：播放下一首
4. **音乐列表**：
   - 上传的音乐会自动加入播放列表
   - 您可以在播放列表中选择要播放的音乐
   - 支持删除不需要的音乐文件

### 2. 图片管理

1. 进入「配置」页面 →「图片管理」
2. **上传图片**：点击「上传图片」按钮，上传您的图片文件
3. **图片使用**：
   - 上传成功后，图片会自动加入图片列表
   - 您可以在「奖品配置」中选择这些图片作为奖品图片
   - 支持删除不需要的图片文件

### 3. 界面配置

1. 进入「配置」页面 →「界面配置」
2. **图案设置**：
   - 在图案设置区域，使用鼠标点击可以自定义配置首页中的高亮图案
   - 您可以根据自己的喜好调整图案的位置和样式
3. **奖项列表显示**：
   - 勾选「是否常显奖项列表」：首页不展示奖品列表
   - 取消勾选：首页会展示奖品列表

## 四、快捷键

在抽奖页面，系统设置了以下快捷键，方便您快速操作：

| 快捷键 | 功能说明 |
| --- | --- |
| **Space**（空格键） | 进入抽奖/开始抽奖/抽取幸运儿/继续抽奖 |
| **Esc**（退出键） | 取消当前操作 |

## 五、数据管理

### 1. 查看已中奖人员

- 进入「配置」页面 →「人员配置」→「已中奖人员」
- 在这里可以查看所有已中奖人员的详细信息
- 包括：人员姓名、部门、获奖奖项、获奖时间等

### 2. 重置数据

- 如果您需要重新开始抽奖，可以在「人员配置」中重置数据
- 注意：重置数据会清空所有中奖记录，但不会删除您的人员和奖品配置

## 六、常见问题

1. **问题**：为什么点击按钮时按钮值不会立即更新？
   **解答**：这是正常现象，按钮值会等动画结束后才会更新为目标值，确保抽奖过程的流畅性。

2. **问题**：如何修改背景音乐？
   **解答**：进入「配置」→「音乐管理」，上传您喜欢的音乐文件，系统会自动将其加入播放列表。

3. **问题**：如何修改抽奖页面的背景图片？
   **解答**：进入「配置」→「图片管理」，上传背景图片，然后在「界面配置」中设置为背景图片。

4. **问题**：抽奖过程中可以修改奖项设置吗？
   **解答**：不建议在抽奖过程中修改奖项设置，以免影响抽奖结果的准确性。
